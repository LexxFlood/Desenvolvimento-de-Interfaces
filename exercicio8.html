Exemplo 8 html - Uso das tags <ol>, <ul> e <li>

As tags <ol>, <ul> representa listas em html e são utilizadas para as mais diversas finalidades. A <ol> representa uma lista ordenada já a tag <ul> representa uma lista não ordenada

Em ambas as listas para representar o selementos da lista usamos a tag <li>

Alem disso estas listas podem conter outras lsitas de forma alinhada.

Etapas para desenvolver um site (lista ordenada)

  1. planejamento
    1. Definir o objetivo do site
    2. Identificar o publico alvo
    3. Criar um mapa do site
  2. design
    1. Criar wireframes e protótipos
    2. Definir a paleta de cores e tipografia
    3. Desenvolver o layout no figma ou adobe xd
  3. desenvolvimento
    1. configurar o ambiente de desenvolvimento
    2. codificar o front-end(html, css e javascript)
    3. Implementar o back-end, com linguagens orientadas a objetos(c#, java, php, python)
  4. testes
    1. testar a responsividade em diferente aparelhos
    2. Realizar testes de usabilidade
    3. corrigir bugs identificados
  5. publicação
    1. escolha um provedor de hospedagem
    2. Configurar o odminio e o ssl
    3. Fazer o deploy do site
