EXEMPLO 11 HTML - FOMULARIOS

os formulario em html sao um dos principais recursos para a coleta de dados, eles devem ser simples e altamente funcionais, atendendo sempre as necessidades dis usuarios.

Para trabalhar com formulario em html, possuimos uma serie de componentes que vao facilitar o processo de interacao com o usuario. Todos os furmularios começam necessariamente com a tag <form> essa tag indica para o navegador que a pagina html possui um formulario que pode enviar dados.

Além da tag <form> temos tambem que usae os atributos action que define o local -url- em que os dados recolhidos/coletados devem ser enviados, e o method que define qual o metodo http para enviar os dados GET ou POST.
