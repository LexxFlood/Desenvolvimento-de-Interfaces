bootstrap

o bootstrap ´e um framework front-end que fornece componentes e ferramentas para facilitar a ciação de siters responsivos. Ele foi desenvolvido pelo twitter e é muito útil para aqueles que precisam de aplicar css a decumentos html.

container

container é uma classe fundamental do bootsrap usada para delimitar e alinhar o conteudo na pagina definindo limites para o layout e garantinfo o espaçamento adequado entre os elemento

O seletor classe container pode ser aplicado de duas formas container ou container-fluid. Sendo assim podemos dizer que o container quando aplicado delimita larguras fixas nas laterais do documento html. já o container fluid ocupa quando aplicado 100% do layout da tela disponível.
