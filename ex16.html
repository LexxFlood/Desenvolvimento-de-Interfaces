Visão Geral sobre CSS

CSS é um acrônimo para folhas de estilo em cascata. Essa técnica visa personalizar, estilizar um documento HTML. Para isso, usamos regras CSS, que representam uma estrutura que define como os elementos selecionados devem ser estilizados. Essa regra é composta pelas seguintes partes:

  • Seletor: Indica quais elementos serão afetados pelo estilo definido.
  • Bloco de Declaração: É o bloco composto por propriedades e valores, esse bloco será aplicado ao elemento HTML.

Tipos de seletores

Em CSS, seletores são padrões usados para selecionar e estilizar os elementos HTML em uma página web. Eles definem quais elementos receberão as regras de estilos que serão declaradas dentro das chaves. Possuímos vários tipos de seletores, os principais são:

Seletores de Classe

Os seletores de classe em CSS são aqueles que são aplicados a elementos HTML que possuam o atributo class. Esse seletor é utilizado amplamente quando queremos estilizar um documento HTML.

Este seletor sempre começará com um ".", seguido pelo nome do seletor e suas propriedades e valores.
