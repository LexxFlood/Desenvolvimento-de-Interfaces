Seletor de ID

O seletor de ID é aquele que aplica a regra CSS a um componente HTML que tenha um atributo id. Esse seletor de ID deve ser aplicado a um único elemento HTML, pois o ID deve ser único. Além disso, esse seletor terá prioridade máxima sobre os demais seletores (de classe e de tipo).

Neste exemplo estamos utilizando seletores de ID para estilizar os elementos do documento HTML.
